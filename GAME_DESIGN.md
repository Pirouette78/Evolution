# GAME DESIGN DOCUMENT
**Concept:** A contemplative tactical evolution game.

## The 4 Official Phases
1. **Cellular (Micro):** Survive in a human body. Eat energy, divide, move randomly.
2. **Tactical Slime:** Macro view of the infection. Strategic placement of "hives".
3. **Zombie (Humanoid):** Conquer the city/planet. Infect humans, gestation logic.
4. **Galactic:** Space conquest, interstellar travel, system-wide domination.

## Gameplay Loop
- **Indirect Control:** Players modify the "Rules of Life" via a Moddable Tech Tree.
- **Persistence:** If a player wins Phase 1, they spawn in Phase 2 on a shared Planet map.
- **Spectator:** Eliminated players can watch any active instance via the Slime Maps.
